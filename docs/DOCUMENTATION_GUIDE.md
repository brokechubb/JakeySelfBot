## Adding New Documentation

When creating new documentation:
1. Place it in this directory
2. Use descriptive file names
3. Include a clear title in the document
4. Organize by functional area when possible

## File Management

- Main project information remains in the root directory (`README.md`, `AGENTS.md`)
- All detailed documentation resides in the `docs/` directory
- This provides clean separation between essential root files and detailed documentation
